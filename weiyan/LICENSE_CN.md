## 一、许可证概览

Apache 许可证 2.0 是一种开源许可证，允许用户：

- 免费使用、复制、修改和再分发本项目
- 用于个人、教育或商业用途
- 创建衍生作品

但需要遵守以下条件。

## 二、主要条款

1. **版权声明**
    - 所有原作者的版权信息必须保留
    - 如果修改了源码，需要在文件或文档中标注修改者和修改日期
    - 可以在原有版权声明后增加你的名字和年份，例如：
      ```
      Copyright 2026 江芷酱紫
      ```

2. **专利许可**
    - 本项目的贡献者自动授予使用者相关专利许可
    - 如果你起诉别人侵犯专利，可能会导致你的专利许可失效

3. **再分发规则**
    - 可以再分发原始或修改后的代码
    - 可以以源代码或二进制形式分发
    - 必须提供 LICENSE 文件副本
    - 修改过的文件需标注修改说明
    - 如果项目包含 NOTICE 文件，也需保留其中的声明

4. **商标使用**
    - 本许可证不授予使用原作者的商标或项目名称的权限
    - 仅可用于合理说明项目来源

5. **免责声明**
    - 项目按 “AS IS” 提供
    - 不提供任何明示或暗示保证
    - 使用者需自行承担风险

6. **责任限制**
    - 任何贡献者不对因使用本项目产生的直接或间接损失负责

## 三、如何应用到你的项目

1. 在项目根目录放置 **LICENSE 文件**，内容为英文原文 Apache 2.0。
2. 在 LICENSE 文件前或 README 中添加版权声明，例如： Copyright 2026 江芷酱紫
3. 可以在源码文件开头加简短版权头：
```cpp

 * Licensed under the Apache License, Version 2.0
 * 本文件遵循 Apache 2.0 许可证，允许使用和修改，但需保留版权信息

```
